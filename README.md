## What is NovelCut

NovelCut turns a novel into a finished short drama.

It is built on the proven shell of an open-source design platform, and the proven soul of an open-source short-drama tool — fused into one tool that does the entire pipeline:

```
novel → events → skeleton → episodes → script → shot list → images → video
```

A node-based canvas keeps every stage visible and editable. A three-layer agent system (decide / execute / supervise) drives generation. A programmable vendor system lets you swap any model — image, video, voice, LLM — without recompiling. Skill files keep prompts editable as Markdown, not buried in code.
